GNU General Public License [Text]
The core of the StreamIt compiler is derived from the Kopi Java compiler, and is consequently released under the GNU General Public License (GPL). However, some of the surrounding code is not subject to the terms of the GPL. All of this release is free to download, modify, and redistribute.

Description of the StreamIt programming language, including all allowed language constructs.

A set of examples demonstrating simple StreamIt programs. The examples show most of the basic StreamIt features, including examples of basic filters, pipelines, split-joins, and feedback loops.

A listing of semantic checks that should be performed by the StreamIt compiler frontend. Many actually are implemented, though not all.
