About This Release

StreamIt README [HTML] [PS] [PDF]
General information about this release of the StreamIt compiler and runtime system. Includes information on dependencies of this package and some warnings on language features that are unimplemented or broken.
Recent Changes (NEWS) [HTML] [PS] [PDF]
High-level listing of changes between releases of the StreamIt compiler and runtime system.
Installation Instructions [HTML] [PS] [PDF]
Installation instructions for the StreamIt compiler and runtime system.
Command-Line Options [HTML] [PS] [PDF]
Listing of command-line options for the knit driver script. Includes options for selecting the uniprocessor, RAW, or Java library backend, as well as various optimization choices.
Test Status [Text]
A listing of the results of automated tests of the compiler and runtime system against the benchmarks in the apps directory.
StreamIt License [Text]
GNU General Public License [Text]
The core of the StreamIt compiler is derived from the Kopi Java compiler, and is consequently released under the GNU General Public License (GPL). However, some of the surrounding code is not subject to the terms of the GPL. All of this release is free to download, modify, and redistribute.

The StreamIt Language

StreamIt Language Specification [HTML] [PS] [PDF]
Description of the StreamIt programming language, including all allowed language constructs.
StreamIt Cookbook [HTML] [PS] [PDF]
A set of examples demonstrating simple StreamIt programs. The examples show most of the basic StreamIt features, including examples of basic filters, pipelines, split-joins, and feedback loops.
Semantic Checks [HTML] [PS] [PDF]
A listing of semantic checks that should be performed by the StreamIt compiler frontend. Many actually are implemented, though not all.

Implementation Notes and Internal Documentation

These documents will not necessarily be useful to the casual user of the StreamIt compiler; however, we distribute them in the hope that people modifying the compiler internals will gain insight into the design decisions and development environment we use.

Implementation Notes [Listing]
A set of text files giving assumptions and design notes for various parts of the StreamIt compiler. Includes some notes on possible implementations for future language features.
Runtime Interface Notes [Listing]
C header files detailing possible implementations for the uniprocessor runtime interface.
CAG StreamIt Manual [HTML] [PS] [PDF]
Instructions for setting up the StreamIt infrastructure in the MIT Computer Architecture Group. This includes a breakdown of the steps taken by the knit script, which may be useful for testing and debugging, along with a description of our methodology for gathering performance statistics.
