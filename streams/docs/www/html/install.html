INSTALLATION



The StreamIt packages are distributed as gziped tar files. Once download, unpack the distributions in a directory of your choice (e.g., tar zxvf streamit.tar.gz). If you downloaded the source StreamIt distribution, you need to build the compiler before using it. To do this, you need to setup your shell enviroment, configure, and build the compiler and libraries. Instructionns are provided below.

  • Review the list of software requirements before proceeding.

  • If you downloaded the binary release, skip to 1b.

1a) Setting the Shell Enviroment (Source Release)

The following environment variables must be set appropriately to build (and subsequently use) the StreamIt compiler.

  • If you are using bash (customize to your system)
    	  ##Customize to system

	  #Where streamit was untarred to
	  export STREAMIT_HOME=/home/quaker/streams

	  #Jikes location
	  export JIKES_BIN_PATH=/usr/bin

	  #Java location
	  export JAVA_BIN_PATH=/usr/java/bin

	  #Java lib locations (where rt.jar is)
	  export JAVA_LIB_PATH=/usr/java/lib

	  #Antlr location
	  export ANTLR_LIB_PATH=/opt/antlr

	  ##Copy rest as is
	  export CLASSPATH=.:${JAVA_LIB_PATH}/rt.jar:${ANTLR_LIB_PATH}/antlr.jar:${STREAMIT_HOME}/src:${STREAMIT_HOME}/3rdparty:${STREAMIT_HOME}/3rdparty/cplex/cplex.jar

          export PATH=$PATH:${STREAMIT_HOME}/bin:${JIKES_BIN_PATH}:${JAVA_BIN_PATH}

	  alias strc=${STREAMIT_HOME}/strc
  • If you are using tcsh (customize to your system)
    	  ##Customize to system

	  #Where streamit was untarred to
	  setenv STREAMIT_HOME  /home/quaker/streams

	  #Jikes location
	  setenv JIKES_BIN_PATH /usr/bin

	  #Java location
	  setenv JAVA_BIN_PATH  /usr/java/bin

	  #Java lib locations (where rt.jar is)
	  setenv JAVA_LIB_PATH  /usr/java/lib

	  #Antlr location
	  setenv ANTLR_LIB_PATH /opt/antlr

	  ##Copy rest as is
	  setenv CLASSPATH .:${JAVA_LIB_PATH}/rt.jar:${ANTLR_LIB_PATH}/antlr.jar:${STREAMIT_HOME}/src:${STREAMIT_HOME}/3rdparty:${STREAMIT_HOME}/3rdparty/cplex/cplex.jar

          set path = ($path ${STREAMIT_HOME}/bin ${JIKES_BIN_PATH} ${JAVA_BIN_PATH})

	  alias strc ${STREAMIT_HOME}/strc

1b) Setting the Shell Enviroment (Binary Release)

If you downloaded the binary release, setup your environment as follows and then test the compiler as shown above:

  • If you are using bash (customize to your system)
    	  #Where streamit was untarred to
	  export STREAMIT_HOME=/home/quaker/streams

	  export CLASSPATH=.:${STREAMIT_HOME}/streamit.jar

          export PATH=$PATH:${STREAMIT_HOME}

	  alias strc=${STREAMIT_HOME}/strc
  • If you are using tcsh (customize to your system)
    	  #Where streamit was untarred to
	  setenv STREAMIT_HOME  /home/quaker/streams

	  setenv CLASSPATH .:${STREAMIT_HOME}/streamit.jar

          set path = ($path ${STREAMIT_HOME})

	  alias strc ${STREAMIT_HOME}/strc

2a) Configuring the Compiler (Source and Binary)

An autoconfiguration script is available in ${STREAMIT_HOME}. To configure the compiler, launch the script from the top level StreamIt directory as follows: 

            cd ${STREAMIT_HOME}
        ./configure
If you downloaded the binary release, skip to 3.

2b) Building the StreamIt Compiler (Source)

Once your system is configured, compile StreamIt as follows: 

    	cd ${STREAMIT_HOME}
        make

3) Testing the StreamIt Compiler (Source and Binary)

Once your system is built, run the following simple test: 

            cd /tmp
        cp $STREAMIT_HOME/apps/examples/hello/HelloWorld6.str .
        strc HelloWorld6.str
        ./a.out -i10
The a.out program should print the numbers 0 through 9, in order, on your screen.
