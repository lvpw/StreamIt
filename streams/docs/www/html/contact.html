CONTACT US



For general information on StreamIt, or to report bugs in the compiler, or to request new language features you may write to streamit@cag.lcs.mit.edu.

You can reach other application developers that are using StreamIt by writing to streamit-users@cag.lcs.mit.edu. To subscribe to the users mailing list, or to review archived postings, please visit the StreamIt users information page.

The mailing list for active StreamIt developers is streamit-dev@cag.lcs.mit.edu. To subscribe to the developers mailing list, or to review archived postings, please visit the StreamIt developers information page.
