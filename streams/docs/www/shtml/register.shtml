The StreamIt compiler is built on top of version 1.5B of the Kopi Java compiler from DMS Decision Management Systems GmbH. The independent components of the system, including the scheduler, Java runtime library, C library, and the frontend, are released subject to the terms of the MIT license agreement. Other components of the system are released under the terms of the GNU General Public License. The binary release also include class files from the ANTLR LL(k) parser generator.
