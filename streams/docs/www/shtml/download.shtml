DOWNLOAD

A public release of the StreamIt compiler, including the StreamIt benchmarks, is available for download. You may download either the full source package which includes all of the StreamIt sources, or the full binary package which does not contain any sources, but includes a Java jar file of the full compiler.

  • Review the list of software requirements before downloading StreamIt.

  • Read the installation instructions after you have downloaded StreamIt.

  • Browse the quick start guide to StreamIt for instructions on using the compiler.

  • For technical assistance and support, you may contact the StreamIt support team.

Eclipse Plugin

An Eclipse Development Plugin allows users to write and debug StreamIt applications in the Eclipse development environment. It is not included in the StreamIt compiler download.
