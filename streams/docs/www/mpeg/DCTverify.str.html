File DCTverify.str
Version 1.0

This file contains functions that check the correctness of various Discrete Cosine Transforms and their inverses. When reference is made to the IEEE DCT specification, it is refering to the IEEE DCT specification used by both MPEG and JPEG. A definition of what makes an 8x8 DCT conform to the IEEE specification, as well as a pseudocode implementation, can be found in Appendix A of the MPEG-2 specification (ISO/IEC 13818-2) on P. 125.
