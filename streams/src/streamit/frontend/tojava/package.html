]> Overview of streamit.frontend.tojava

Helper classes for converting StreamIt code to Java. These classes are used by conversion classes like streamit.frontend.ToJava to convert the code. {@link streamit.frontend.tojava.NodesToJava} takes an IR tree and produces a Java source file from it; this is the last stage in conversion.
