]> Overview of streamit.frontend.nodes

Intermediate representation nodes for the StreamIt frontend. This package contains a high-level representation of a StreamIt program. Nodes in this package are generated by {@link streamit.frontend.StreamItParserFE}. An entire program is stored in a {@link streamit.frontend.nodes.Program} object, which contains a list of {@link streamit.frontend.nodes.StreamSpec} objects. Each of these objects in turn contains the actual code making up the StreamIt program.

By and large, the IR objects here are immutable. The {@link streamit.frontend.nodes.FEVisitor} class implements the visitor pattern, which trades class polymorphism for method polymorphism. (See Design Patterns by Gamma, Helm, Johnson, and Vlissides.) Thus, most classes in this package have an accept method, which accepts a visitor object and calls the appropriate method on it.
