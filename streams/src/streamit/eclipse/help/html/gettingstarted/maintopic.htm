Installing Eclipse

 

  • Install Sun Java Runtime Environment j2re-1_4_2_03

(IBM Java is not compatible; the XML library fails)

  • Install Eclipse eclipse-DKS-2.1.1
  • Install GEF GEF-runtime-I20030618

 

Installing the Debugger

 

  • Launch Eclipse
  • From within Eclipse, click on WINDOWS then PREFERENCES
  • Under WORKBENCH, unselect "PERFORM BUILD AUTOMATICALLY ON RESOURCE MODIFICATION"
  • Under PLUG-IN DEVELOPMENT->TARGET PLATFORM, select "ANOTHER COMPATIBLE APPLICATION INSTALLED AT THE FOLLOWING LOCATION"  (the APPLICATION LOCATION should match the location of your Eclipse)
  • Click on the SELECT ALL button (below the BROWSE and RELOAD buttons)
  • Click on OK to dismiss the window
  • From within eclipse, Click on FILE then IMPORT
  • Select EXISTING PROJECT INTO WORKSPACE
  • Click on BROWSE and find "streamit" (the location of this directory depends on where you unpack the plugin)
  • Click on FINISH
  • Select EXISTING PROJECT INTO WORKSPACE
  • Click on BROWSE and find "streamit.eclipse.debugger"  (the location of this directory depends on where you unpack the plugin)
  • Click on FINISH

 

Running the Debugger

 

  • Click on RUN->RUN AS->RUN-TIME WORKBENCH
